# Grokayem Algosobesy: Скользящее окно

Паттерн Скользящее окно (Sliding Window) — мощный инструмент для эффективного решения задач на непрерывные подмассивы и подстроки за линейное время. Он особенно полезен, когда нужно оптимизировать поиск с ограничениями: сумма ≥ X, все символы уникальны, не более K типов и т.п.

## Содержимое
- [`Grokaem_algosobesy_Skolzyashee_okno.pdf`](Grokaem_algosobesy_Skolzyashee_okno.pdf) — подробная презентация: определение, два типа окна (фиксированное и переменное), визуализация, сравнение с наивным подходом, примеры задач.
- [`fix_window.py`](fix_window.py) — реализация задачи Maximum Sum Subarray of Size K (фиксированное окно).
- [`209.py`](209.py) — решение задачи [209. Minimum Size Subarray Sum](https://leetcode.com/problems/minimum-size-subarray-sum/).

## Когда использовать?
Применяется, когда:
- Нужно найти непрерывный подмассив или подстроку в исходных данных
- Есть ограничение: сумма ≥ X, длина = k, все символы уникальны, не более K различных элементов
- Требуется максимизировать, минимизировать или найти все подходящие варианты

Не подходит, если:
- Подмассив может быть разрывным (тогда используйте динамическое программирование)
- Задача про перестановку без учёта порядка элементов
- Требуется отслеживать всю историю предыдущих значений или состояний
