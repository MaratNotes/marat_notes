# Grokayem Algosobesy: Два указателя

Разбор паттерна **«Два указателя»** (Two Pointers) в рамках серии «Grokayem Algosobesy».

## Содержимое
- **`Grokaem-algosobesy_Two_Pointers.pdf`** — презентация с объяснением идеи, типов задач и визуализацией работы алгоритма.
- **`decision.py`** — реализация решения задачи [LeetCode 167. Two Sum II – Input Array Is Sorted](https://leetcode.com/problems/two-sum-ii-input-array-is-sorted/).

## Задачи для практики (по возрастанию сложности)
1. [**125. Valid Palindrome**](https://leetcode.com/problems/valid-palindrome/description/) — проверка палиндрома с помощью двух указателей с двух концов строки.  
2. [**11. Container With Most Water**](https://leetcode.com/problems/container-with-most-water/description/) — классическая задача на максимизацию площади между двумя вертикальными линиями.  
3. [**15. 3Sum**](https://leetcode.com/problems/3sum/description/) — продвинутый пример: комбинирование «двух указателей» с сортировкой для поиска троек чисел.
