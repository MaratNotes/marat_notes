# Разворот связного списка на месте  
Грокаем алгособесы #5

Решение классической задачи с алгоритмического собеседования: как развернуть односвязный список на месте за O(n) времени и O(1) дополнительной памяти.

## Формулировка задачи
Дан указатель head на начало односвязного списка. Требуется развернуть список на месте и вернуть указатель на новое начало.

Требования:
- Время выполнения: O(n)
- Дополнительная память: O(1)

Пример:  
Вход: 1 → 2 → 3 → 4 → null  
Выход: 4 → 3 → 2 → 1 → null

Ссылка: LeetCode #206 — Reverse Linked List  
https://leetcode.com/problems/reverse-linked-list/

## Ключевые идеи
- Используются три указателя: prev, curr, next_temp
- Сначала сохраняется next_temp, затем изменяется curr.next
- После завершения цикла возвращается prev — он становится новой головой списка

## Типичные ошибки
- Изменение curr.next до сохранения next_temp приводит к потере остатка списка
- Возврат curr вместо prev — к концу цикла curr равен null
